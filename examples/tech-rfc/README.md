# Technical RFC Example

This example demonstrates a technical RFC presentation for infrastructure upgrades.

## Files

- `tech-rfc.mu` - The Mockup source file (6 slides)

## Content

1. Cover: Database Infrastructure Upgrade Proposal
2. Current state and problems (with emoji indicators)
3. Target architecture diagram
4. Auto-scaling strategy with line graph
5. Migration plan timeline
6. Expected benefits

## Generated Images

Generated presentation images from different AI models will be placed here:

- `claude-sonnet-4/` - Images generated by Claude Sonnet 4
- `gpt-4/` - Images generated by GPT-4
- `other-models/` - Images from other AI models

Each subdirectory will contain:
- Individual slide images (`slide-01.png`, `slide-02.png`, etc.)
- Full presentation PDF if available
- Generation metadata (model version, date, parameters)
