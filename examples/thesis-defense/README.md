# Thesis Defense Example

This example demonstrates an academic thesis defense presentation.

## Files

- `thesis-defense.mu` - The Mockup source file (7 slides)

## Content

1. Title slide with paper metadata
2. Problem statement with comparison diagrams
3. Model architecture (U-Net structure)
4. Quantitative results with bar chart
5. Qualitative results with 4-column comparison
6. Summary with double-line border emphasis
7. Acknowledgments

## Generated Images

Generated presentation images from different AI models will be placed here:

- `claude-sonnet-4/` - Images generated by Claude Sonnet 4
- `gpt-4/` - Images generated by GPT-4
- `other-models/` - Images from other AI models

Each subdirectory will contain:
- Individual slide images (`slide-01.png`, `slide-02.png`, etc.)
- Full presentation PDF if available
- Generation metadata (model version, date, parameters)
