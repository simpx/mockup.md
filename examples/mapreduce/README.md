# MapReduce Paper Presentation Example

This example demonstrates a presentation for the classic MapReduce paper (Google, 2004).

## Files

- `mapreduce.mu` - The Mockup source file (15 slides)

## Content

Complete teaching presentation covering:

1. Title and author information
2. Motivation and background
3. Programming model overview
4. Example: Word Count
5. Execution overview with architecture diagram
6. MapReduce workflow sequence diagram
7. Master data structures
8. Fault tolerance mechanisms
9. Locality optimization
10. Task granularity considerations
11. Backup tasks strategy
12. Performance results with bar charts
13. Large-scale production use cases
14. Related work comparison
15. Summary and conclusions

## Generated Images

Generated presentation images from different AI models will be placed here:

- `claude-sonnet-4/` - Images generated by Claude Sonnet 4
- `gpt-4/` - Images generated by GPT-4
- `other-models/` - Images from other AI models

Each subdirectory will contain:
- Individual slide images (`slide-01.png`, `slide-02.png`, etc.)
- Full presentation PDF if available
- Generation metadata (model version, date, parameters)
